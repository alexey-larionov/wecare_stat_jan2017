filter_cases_wecare_jan2017

started: Alexey Larionov, 01Mar2016
last updated: Alexey Larionov, 26Jan2017

Summary

This is the last step that prepares dataset for downstream analyses.

It performs the following:
- remove cases, which failed sequencing

  • remove variants that were present in the removed cases only

  • explore the remaining data

Exac and kgen include only variants that were biallelic in exac/ kgen respectively
